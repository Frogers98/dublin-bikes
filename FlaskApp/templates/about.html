Stations About Disappster

Welcome to our application!

Thanks for taking the time to check out Disappster, a state-of-the-art solution for planning your journeys around Dublin City Centre, which integrates the power of Machine Learning Models and hyperparameter tuning to help you choose a sustainable method of exploring 5km of Dublin in these uncertain times!

Completed as part of the COMP30830 Software Engineering module, this application sees to all of the needs of the modern bike commuter. Meeting requirements such as:

Key Name Description
1 JCDecaux Data Collects data through JCDecaux API
2 Data Managment data management/storage in RDS DB on AWS
3 A Map display bike stations on map
4 Flask-based flask web application (python)
5 Occupancy Station Occupancy information
6 weather Weather information
7 Interactivity Interactivity (click, API request, handle response)
8 Machine Learning Models ML model for predicting occupancy based on weather patterns, trained on collected data
9 Github source-coded Github repo, including source code, logs, commit history, branches, etc

this application is hopefully sure to impress with an industrial-level Jira and Confluence board only a click away for even more exploration into Disappster.
